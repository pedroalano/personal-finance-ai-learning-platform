# Personal Finance AI Learning Platform

An AI-powered platform designed to help users learn and manage personal finance through interactive experiences.

## Project Structure

This is a TypeScript monorepo using npm workspaces.

```
.
├── apps/
│   ├── api/        # Backend API server
│   └── web/        # Frontend web application
└── packages/
    └── shared-types/  # Shared TypeScript type definitions
```

## Prerequisites

- Node.js 18+
- npm 9+

## Getting Started

Install dependencies from the root:

```bash
npm install
```

## Development

Run each app from its directory or add workspace scripts as needed.

## Tech Stack

- **Language**: TypeScript
- **Linting**: ESLint + Prettier
- **Package manager**: npm workspaces
